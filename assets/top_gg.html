Russian Spy Bot  

  General:  

This Discord bot logs some interesting information about users in your guild. A user is considered as active if he is in a voice channel and not muted. If he is muted, he is considered as afk. The bot also counts how many messages a user sends per day. This bot encourages members to compare themselves with others and to be more active on the Discord server.

If you have any other trouble with the bot or ideas for improvement just direct message the bot and our mod team will respond as soon as possible.

  Configuration:  

  1. Invite the bot to your discord server with link above or right here.
  2. Set a custom prefix with !setting prefix .
    • You need to be a server administrator to change the prefix. The default prefix is ! .
  3. Get commands to display your stats with !help .
    • Quick note: You can not see any stats from times before you added the bot.
  4. You can set your time zone with !timezone . The default time zone is the Coordinated Universal Time (UTC).

  Commands:  

Command Description
!help Sends you a list with all available commands. By adding a command name behind you can get more information about the command.
!stats Sends information about your active time, afk time and messages sent.
!timezone Set your new time zone, which is needed for more accurate stats.
!leaderboard Sends a Leaderboard for the total stats with the top five users of the server you are currently on.
!average Sends back your average statistics for the last ten days.
!settings Gives you options to configure the bot for your server.

  Security:  

This bot is run on a secure online server and the owner guarantees that it is safe. The user data is stored in a database which is secured against SQL injection. This way only the user can access his own data and no one else.

  About the name:  

This name is meant as a joke and has nothing to do with Russians. The bot is spying trying to help you and he swore to never give out any of your information to any strangers. If you have a big problem with the name, you can set a custom nickname for your server just by right clicking the bot.
